# Testimonial Images Setup

To display the testimonial images, add these 3 image files to this `public` folder:

1. **sarah.jpg** (or .jpeg/.png)
   - Sarah, 28, Marketing
   - Woman with short dark curly hair, pink shirt, navy blazer

2. **marcus.jpg** (or .jpeg/.png)
   - Marcus, 34, Former Corporate Drone
   - Man with light brown hair, beard, navy sweater, in car

3. **jamie.jpg** (or .jpeg/.png)
   - Jamie, 31, Overthinker in Recovery
   - Woman with red hair and freckles, blue dress, patterned scarf

The code will automatically try different file extensions (.jpg, .jpeg, .png, .webp) if the first one doesn't work.
